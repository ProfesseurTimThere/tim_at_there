GUIDE 3 : LE TABLEUR AVANCE

OBJECTIF : Analyser et synthetiser de gros volumes de donnees (Fonctions Conditionnelles, TCD)

Ce guide s'adresse aux eleves qui maitrisent les bases (Guide 2). Ici, nous transformons les donnees brutes en informations strategiques, indispensables pour les rapports techniques et de gestion.

1. LE CONDITIONNEL : Le Pouvoir du "SI"

Ces formules permettent d'analyser vos donnees uniquement si elles remplissent un ou plusieurs criteres precis.

1.1 Fonction =SI() (Le Test Logique)

Cette fonction est la base de toute prise de decision automatisee dans un tableau.

  • Role : Tester si une condition est remplie.
  • Syntaxe : =SI(test_logique; valeur_si_vrai; valeur_si_faux)
  • Exemple : Vous voulez savoir si l'objectif de vente est atteint (colonne A > 50). =SI(A2>50; "Objectif atteint"; "A revoir")

1.2 Fonctions de Comptage et de Somme Conditionnelles

FONCTION ROLE SYNTAXE EXEMPLE
=NB.SI(plage; critere) Compte le nombre de cellules d'une plage qui respectent un seul critere. =NB.SI(D:D;"Bovins") (Compte le nombre de bovins)
=SOMME.SI(plage; critere; plage_somme) Additionne les valeurs d'une plage si un seul critere est respecte. =SOMME.SI(B:B;"Male"; C:C) (Somme des poids des males)

2. SYNTHESE RAPIDE : Tableaux Croises Dynamiques (TCD)

Le TCD est l'outil le plus puissant pour synthetiser des milliers de lignes de donnees en quelques secondes. Il est indispensable pour tout rapport d'analyse.

2.1 Creation et Utilisation

  1. Preparation : Assurez-vous que vos donnees sont au format "Tableau" (voir Guide 2) et qu'il n'y a aucune ligne vide ou fusionnee.
