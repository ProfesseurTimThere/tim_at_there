I. LA RÈGLE D'OR : CODEX DU DOCUMENT

Règle d'Or - Arrêtez les ENTER et ESPACES

Auteur : Tim@There Objectif : Adopter la mentalité du professionnel pour créer un document stable et automatisable.

Votre document est un code. Si vous utilisez les touches ENTER et ESPACE au lieu des outils d'automatisation, il se déformera à la moindre modification !

❌ CE QUE FONT LES AMATEURS ✅ CE QUE FONT LES PROS LA RÈGLE D'OR (À MAÎTRISER)
Pousser le texte avec ENTER (ligne par ligne) pour sauter de page. Utiliser le Saut de Page. Pour commencer une nouvelle page : Onglet Insertion > Saut de Page.
Mettre des ESPACES pour aligner un mot ou un bloc. Utiliser les Tabulations ou la Justification. Pour aligner (carré) : Onglet Accueil > Justification.
Changer l'orientation de tout le document pour un tableau large. Utiliser un Saut de Section. Pour changer l'orientation d'une seule page : Onglet Mise en Page > Sauts > Page Suivante.

II. Les Fondamentaux du Texte

Fiche Outil - Sélection et Alignement du Texte

Auteur : Tim@There Objectif : Maîtriser la sélection rapide et les alignements de base.

Action Étapes Clés (Générique) Raccourci / Astuce
Sélectionner tout le document Non applicable. CTRL + A (Windows) ou CMD + A (Mac)
Sélectionner un mot / paragraphe Double-cliquer le mot / Triple-cliquer le paragraphe. Astuces pour cibler rapidement un bloc.
Aligner le texte (Gauche, Centre, Droite, Justifié) 1. Sélectionnez le texte ou placez le curseur dans le paragraphe. 2. Dans l'Onglet Accueil (ou Format de Texte), cherchez le groupe Paragraphe.

Fiche Outil - Police et Attributs de Caractères

Auteur : Tim@There Objectif : Appliquer les mises en forme des caractères, la couleur et les effets.

Action Étapes Clés (Générique) Localisation Générique
Modifier la Police (fonte) 1. Sélectionnez le texte. 2. Dans l'Onglet Accueil (ou Format de Texte), cherchez le groupe Police et utilisez le menu déroulant.
Appliquer Gras / Italique / Souligné 1. Sélectionnez le texte. 2. Dans l'Onglet Accueil, cliquez sur les icônes G, I, ou S.
Utiliser Exposant/Indice 1. Sélectionnez le caractère (ex: le 2 de H₂O ou le ᵉ de 1ᵉʳ). 2. Dans le groupe Police, cliquez sur l'outil Indice (X₂) ou Exposant (X²).
Couleur du texte et de l'arrière-plan 1. Sélectionnez le texte. 2. Dans le groupe Police, utilisez l'outil Couleur de la police (A) pour le texte. Utilisez l'outil Surligneur ou l'option Trame de fond pour le fond du paragraphe.
Effacer toute la mise en forme Sélectionnez le texte, puis utilisez l'outil Effacer la mise en forme (souvent une gomme avec un A) dans le groupe Police. Utile pour repartir de zéro.

III. Structurer la Page

Fiche Outil - Mise en Page Simple et Saut de Section

Auteur : Tim@There Objectif : Définir la structure de base du document (marges, orientation, taille).

Action Étapes Clés (Générique) Utilisation Stratégique
Définir les Marges 1. Allez dans l'Onglet Mise en Page (ou Format / Disposition). 2. Cliquez sur l'outil Marges et choisissez un modèle ou Marges personnalisées.

Insérer un Saut de Section Allez dans l'Onglet Mise en Page > Sauts > Saut de Section Continu ou Page Suivante. Le Saut de Section vous permet d'avoir des zones de mise en page indépendantes dans le même document.

Ligne séparatrice entre les colonnes 1. Allez dans Autres colonnes.... 2. Cochez l'option Ligne séparatrice.
Insérer une Lettrine 1. Sélectionnez la première lettre du paragraphe. 2. Allez dans l'Onglet Insertion.

Fiche Outil - Les Tableaux : Création et Gestion

Auteur : Tim@There Objectif : Créer et manipuler une structure de données.

Action Étapes Clés (Générique) Concepts Clés
Créer un tableau simple 1. Allez dans l'Onglet Insertion. 2. Cliquez sur l'outil Tableau et dessinez la taille.
Fusionner/Fractionner des cellules 1. Sélectionnez les cellules à modifier. 2. Utilisez le clic-droit ou l'Onglet Disposition des Outils Tableau.

VI. Éthique et Références

Règles d'Or et Intégrité Scientifique

Auteur : Tim@There Objectif : Comprendre et appliquer les règles d'honnêteté intellectuelle dans tout travail.

Règle d'Or Principe Conséquences
Citer ses sources Toute idée, phrase, image, ou donnée qui n'est pas la vôtre doit être attribuée à son auteur original. Utilisation de l'outil de Citations et Bibliographie (Onglet Références).
Vérifier l'information Ne jamais recopier une information sans en vérifier la crédibilité (date, auteur, source). Vos travaux doivent reposer sur des faits et non sur des opinions non étayées.
Présenter un travail original Le plagiat (copier sans citer) est une faute grave. Votre travail doit être une production personnelle. Respect de l'honnêteté intellectuelle.
Respecter la Licence Creative Commons Quand vous utilisez des images ou des documents trouvés en ligne, vérifiez si vous avez le droit de les utiliser (Licence Libre de droits). Éviter les problèmes de droits d'auteur.
