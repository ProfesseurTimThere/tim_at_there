I. LA RÈGLE D'OR : CODEX DU DOCUMENT

Règle d'Or - Arrêtez les ENTER et ESPACES

Auteur : Tim@There Objectif : Adopter la mentalité du professionnel pour créer un document stable et automatisable.

Votre document est un code. Si vous utilisez les touches ENTER et ESPACE au lieu des outils d'automatisation, il se déformera à la moindre modification !

❌ CE QUE FONT LES AMATEURS ✅ CE QUE FONT LES PROS LA COMMANDE DE SURVIE (Outil)
Pousser le texte avec ENTER (ligne par ligne) pour sauter de page. Utiliser le Saut de Page. Onglet Insertion > Saut de Page.
Mettre des ESPACES pour aligner un mot ou un bloc. Utiliser les Tabulations ou la Justification. Onglet Accueil > Justification (pour un alignement carré).
Changer l'orientation de tout le document pour un tableau large. Utiliser un Saut de Section. Onglet Mise en Page > Sauts > Page Suivante (pour rendre la page indépendante).

II. Les Fondamentaux du Texte

Les Outils de Base du Texte

Objectif : Mettre en forme rapidement le texte pour la lisibilité.

Action de Survie Étapes Clés (Générique) Conseil du Pro
Sélectionner un paragraphe Triple-cliquer n'importe où dans le paragraphe. La sélection est la première étape de toute modification.
Appliquer Gras / Italique Sélectionnez le texte > Onglet Accueil > Cliquez sur G ou I. Utilisez CTRL + G et CTRL + I pour aller plus vite.
Utiliser Exposant/Indice Sélectionnez le caractère (ex: le 2 de H₂O) > Onglet Accueil > Outil Exposant (X²) ou Indice (X²). Indispensable pour les formules (chimie, math).
Nettoyer le format (Urgence) Sélectionnez le texte, puis utilisez l'outil Effacer la mise en forme (la gomme avec un A). À utiliser si le texte a été copié du web et est illisible.

III. Structurer la Page

Mise en Page, Colonnes et Tableaux (Les Indispensables)

Objectif : Définir la structure du document et organiser les données.

Compétence Outil de Survie Étapes Clés
Marges du Document Marges Personnalisées Onglet Mise en Page (ou Disposition) > Marges. Définir le blanc autour du contenu.
Orientation (Page Unique) Saut de Section Pour qu'une seule page soit en mode Paysage : Onglet Mise en Page > Sauts > Saut de Section. Puis changez l'orientation.
Mettre en Colonnes Outil Colonnes Sélectionner le texte (ou la section) > Onglet Mise en Page > Colonnes.
Tableaux (Création) Insertion Onglet Insertion > Tableau. Pour organiser les données en lignes/colonnes.
Tableaux (Modification) Fusionner / Fractionner Sélectionnez les cellules > Clic-droit ou Onglet Disposition des Outils Tableau.

IV. Insérer des Éléments Visuels

Images, Schémas et Objets

Objectif : Placer, redimensionner et habiller correctement tout élément graphique.

Compétence Outil de Survie Conseil du Pro
Redimensionner une Image Poignées de coin Tirez toujours par les poignées de coin pour conserver les proportions et éviter la déformation.
Déplacer l'Image Habillage du Texte Après l'insertion, choisissez une option d'Habillage (Carré ou Rapproché) pour pouvoir déplacer l'image librement.
Insérer un Schéma SmartArt Onglet Insertion > SmartArt. Permet de créer des listes, processus ou cycles clairs.
Équations Outil Équation Onglet Insertion > Équation. Pour intégrer des formules mathématiques complexes et standardisées.
Gestion des Plans Avancer / Reculer Onglet Format. Si deux objets se chevauchent, utilisez ces outils pour choisir celui qui est "par-dessus".

V. Outils du Document Long

Automatisation et Référence (Rapports, Mémoires)

Objectif : Créer une structure automatisée et professionnelle.

Compétence Outil de Survie Étapes Clés (Processus)
Hiérarchiser le Plan Les Styles 1. Sélectionner un titre. 2. Onglet Accueil > Appliquer Titre 1, Titre 2, etc. Le logiciel mémorise la structure.
Générer la Table des Matières Outil Table des Matières 1. Placer le curseur. 2. Onglet Références > Table des Matières. Génération automatique à partir des Styles.
Page de Garde Outil Page de Garde Onglet Insertion > Page de Garde. Elle se place automatiquement au début et n'est pas numérotée.
Citations et Bibliographie Gérer les Sources 1. Onglet Références > Gérer les sources (enregistrer les sources). 2. Insérer la Bibliographie à la fin du document.

VI. Éthique et Intégrité Scientifique

Les Fondamentaux du Respect Intellectuel

Objectif : Comprendre et appliquer les règles d'honnêteté intellectuelle dans tout travail.

Règle d'Or Principe Conséquences
Citer ses sources Toute idée, phrase, image ou donnée qui n'est pas la vôtre doit être attribuée à son auteur original. Utiliser l'outil de Citations et Bibliographie (Onglet Références).
Vérifier l'information Ne jamais recopier une information sans en vérifier la crédibilité (auteur, date, source). Vos travaux doivent reposer sur des faits étayés.
Présenter un travail original Le plagiat (copier sans citer) est une faute grave. Votre travail doit être une production personnelle, même si elle se base sur des recherches. Respect de l'honnêteté intellectuelle.
